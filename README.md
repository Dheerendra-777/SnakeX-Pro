# 🐍 SnakeX Pro

**SnakeX Pro** is a modern, feature-rich arcade Snake game built using **HTML, CSS, and JavaScript**.
It offers a smooth gameplay experience with advanced features like power-ups, themes, leaderboard, team mode, and dynamic difficulty.

---

## 🎮 Features

* 🎨 Multiple Themes (Dark, Light, Retro Arcade)
* ⚡ Difficulty Levels (Beginner, Intermediate, Professional)
* 🧩 Multiple Map Layouts (Classic, Box, Tunnel, Maze, Cross)
* 🍎 Advanced Power-Up System
* 🏆 Leaderboard System (LocalStorage based)
* 👤 Individual Mode & 👥 Team Competition Mode
* 🔊 Sound Effects System
* 📱 Mobile Controls + Touch Support
* 🚧 Dynamic Obstacles Generation
* 📈 Level Progression System
* 🔥 Combo Score System
* 🖥️ Fullscreen Gameplay Experience

---

## 🕹️ How to Play

1. Open the game in your browser
2. Select your preferred settings
3. Click **▶ Play Fullscreen**

### 🎮 Controls

* ⬆️⬇️⬅️➡️ Arrow Keys OR
* ⌨️ WASD Keys
* ⏸️ Spacebar → Pause / Resume

### 🎯 Goal

Eat food to grow your snake and increase your score.
Avoid:

* Walls
* Obstacles
* Your own tail

## ⚡ Power-Ups

| Power-Up        | Description                    |
| --------------- | ------------------------------ |
| 🟡 Golden Apple | Gives 4x points                |
| 🔵 Blue Crystal | Slows down the game            |
| 🟣 Purple Orb   | Attracts food toward the snake |
| ☠️ Poison       | Reduces score                  |
